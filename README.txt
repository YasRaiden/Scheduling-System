Directions to run program-
On start of program login screen is presented to end user. User will enter valid credentials to access scheduling
application. First tab presented will is the schedule with ADD, MODIFY, and DELETE options for modifying data listed in
database. Second Tab will present customers with ADD, MODIFY, and DELETE options for modifying data listed in database.
Additionally tab contains an Add Appointment button to create an appointment directly from the customer record. Reports
button is available from top left which allows access to reporting information in a separate window.
